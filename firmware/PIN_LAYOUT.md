# Pin layout
---
## IR
| **Pin** | **Function** |
| ---- | ---- |
| D3 | Transmitter |
| D7 | Receiver |

## 433MHz RF
| **Pin** | **Function** |
| ---- | ---- |
| D8 | Transmitter |
| D2 | Receiver |

## LED Strip
| **Pin** | **Function** |
| ---- | ---- |
| D4 | LED input |

## Buzzers
| **Pin** | **Function** |
| ---- | ---- |
| D5 & D6 | Buzzer 1 input |
| D9 & D10 | Buzzer 2 input |

## Buttons
| **Pin** | **Function** |
| ---- | ---- |
| A2 | Mode switch |
| A3 | Slot select |
| A6 | Learn  |
| A7 | Play |
